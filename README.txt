== Description ==

<strong>WP Reviewr</strong> is a free reviewing plugin version. With this plugin you can add reviews to any posts, pages and custom post types.

This amazing plugin comes with responsive layout that will adapt to any devices resolution such as smartphones and tablets. Easily customize the color scheme using the provided color pickers to match the design to your current theme.

<h4>Features:</h4>
* Unlimited Colors
* Unlimited Review Criterias
* Schema SEO Rich Snippet Review Microdata
* Shortcode for reviews 
* Retina Ready!
* Responsive Layout

Use [reviewr] if you want to manually add the review on the posts.

<h4>Looking for more?</h4>

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. That's it :), Additional <strong>Settings > WP-Reviewr Settings</strong> will be added.

== Frequently Asked Questions ==

For more information,refer to <a href="http://wp-reviewr.com/">WP Reviewr</a> or ask via forum and I will be very happy to help you.

1. How to activate reviews on custom post type? A: Go to "WP-Reviewr Settings" and check the custom post type you want.
2. How to add manually via shortcode? Use <strong>[reviewr]</strong> shortcode.
